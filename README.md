# 🛡️ Esquema Nacional de Seguridad (ENS) - Grafo Interactivo

> **Visualización interactiva del Real Decreto 311/2022** que regula el Esquema Nacional de Seguridad en España.

---

## 📌 Descripción

Este proyecto contiene un **grafo interactivo** que representa la estructura completa del **Esquema Nacional de Seguridad (ENS)**, según el **Real Decreto 311/2022, de 3 de mayo**. El grafo permite explorar de manera visual y dinámica:

- **7 Principios Básicos** que rigen la seguridad de la información en el sector público.
- **16 Requisitos Mínimos** para la protección adecuada de sistemas y datos.
- **Medidas de Seguridad** organizadas en tres marcos (organizativo, operacional y protección de activos).
- **Entidades clave** involucradas en la implementación y supervisión del ENS.
- **Normativa alineada** con el ENS (RGPD, Leyes 40/2015, 39/2015, Directiva NIS, etc.).

1. Clona este repositorio:
  ```bash
   git clone https://github.com/jorgeswork/ens-grafo-web.git
   cd ens-grafo-web
  ```
2. Abre el archivo `index.html` en tu navegador.

## 🎨 Funcionalidades

- **Interacción con el grafo**: Haz clic en los nodos para expandir/collapsar secciones.
- **Cambiar orientación**: Botón para alternar entre vista vertical y horizontal.
- **Exportar**: Descarga el grafo como **PNG** o **SVG**.
- **Reiniciar vista**: Vuelve al estado inicial del grafo.
