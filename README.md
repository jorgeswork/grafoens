# 🛡️ Esquema Nacional de Seguridad (ENS) - Grafo Interactivo

> **Visualización interactiva del Real Decreto 311/2022** que regula el Esquema Nacional de Seguridad en España.

## 📌 Descripción

Grafo interactivo del **Esquema Nacional de Seguridad (ENS)** según el **Real Decreto 311/2022**. Incluye principios, requisitos, medidas, entidades y normativa.

## 🚀 Cómo Usar

### GitHub Pages
🔗 **[https://jorgeswork.github.io/grafoens/](https://jorgeswork.github.io/grafoens/)**

### Local
1. Clona el repositorio: `git clone https://github.com/jorgeswork/grafoens.git`
2. Abre `index.html` en tu navegador.

## 🎨 Funcionalidades

- **Interacción**: Haz clic en nodos para expandir/collapsar.
- **Zoom**: Usa la rueda del ratón o los botones.
- **Arrastre**: Mantén presionado el botón izquierdo para mover el grafo.
- **Exportar**: Descarga como PNG o SVG.
